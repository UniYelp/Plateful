Allows developers to build mobile apps for iOS and Android using JavaScript and [[Architecture/Stack/React]]. It provides near-native performance while letting teams share a large portion of code across platforms, which reduces development time and cost. Developers often pick React Native when they want a single codebase, fast prototyping, and access to both web and mobile ecosystems.
