A modern, type-safe form library for [[React]] that emphasizes type inference, composability, and minimal boilerplate. It provides a predictable way to manage input state, validation, and submission while leveraging [[TypeScript]] for end-to-end type safety.

It fits naturally into the [[TanStack]] ecosystem, and is a strong foundation for building reliable, type-safe forms in our application.
