A fully type-safe router for [[Architecture/Stack/React]] that emphasizes performance, nested routing, and data loading. It integrates tightly with React’s ecosystem and provides a developer-friendly API with strong [[Architecture/Stack/TypeScript]] support. Developers choose TanStack Router for fine-grained control over navigation, first-class TypeScript safety, and reliable handling of complex route hierarchies. Integrates well with the rest of the [[TanStack]] exosystem.
