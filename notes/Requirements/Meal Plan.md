A meal plan is a set of [[Recipes]], set to be cooked over a period of time.

1. users should be able to create, manage and delete meal plans.
2. users should be able to modify parts of the meal plan - for example, changing one recipe without changing the others.
3. view meal plan over a calendar or time view

Creation Flow:
1. the user should be able to add both existing and new recipes when creating the meal plan
2. the user should build the plan with a calendar view
3. the user should be able to choose tags for the entire meal plan - like vegan, healthy, low effort etc.
4. the user should be able to choose whether all the ingredient are allowed to not be in stock or not 
5. In the calendar view, only a card view of the recipes should be shown with all the important details
6. the user should be able to drag and drop the meals over the calendar
7. when done, the user will be shown a success message and forwarded to the meal plan page
