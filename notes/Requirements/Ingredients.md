Ingredients are representation of physical ingredients the user has at home or can buy in order to cook food.

1. the user should be able to see the existing ingredients in his home, including the title, a short description, amount, time it was added, expiry / best before date
2. the ingredient representation should include a picture / drawing representing the ingredient
3. the ingredient title should be unique. 
4. if 2 very similar ingredients are entered, the user should be prompted whether to update the amount of the original or create a new ingredient
